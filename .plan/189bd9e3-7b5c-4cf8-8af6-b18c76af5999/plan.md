# 粒子生命模拟系统 - 项目规划

## 项目概述

实现一个基于 Web 的粒子生命模拟系统（Particle Life Simulation），通过简单的物理规则让粒子产生复杂的类生命行为。该系统模拟不同类型的粒子之间的吸引和排斥关系，形成自组织的有机结构。

### 核心概念
粒子生命模拟基于以下原理：
- 每种粒子类型对其他类型有特定的吸引或排斥力
- 粒子在力的作用下移动，形成动态平衡
- 简单规则产生复杂的群体行为（涌现现象）

## 技术栈

- **框架**: React 19 + Vite
- **渲染**: HTML5 Canvas 2D Context（高性能粒子渲染）
- **样式**: Tailwind CSS + shadcn/ui
- **动画**: requestAnimationFrame 驱动的主循环
- **状态管理**: React hooks (useState, useRef, useCallback)

## 功能需求

### 核心模拟功能
1. **粒子系统**
   - 8 种粒子类型，每种有**完全独特的视觉效果**
   - 2500 个粒子（平衡性能和效果）
   - 粒子间相互作用力计算

2. **粒子类型视觉效果**（每种类型独特）
   - **类型1（红色）**: 圆形，中等大小，强发光，脉冲动画
   - **类型2（橙色）**: 方形，较大，暖光晕，旋转效果
   - **类型3（黄色）**: 星形，小尺寸，闪烁效果，高亮度
   - **类型4（绿色）**: 六边形，中等，呼吸动画，柔和光晕
   - **类型5（青色）**: 菱形，较小，流动拖尾，冷色调发光
   - **类型6（蓝色）**: 圆形带环，大尺寸，涟漪扩散效果
   - **类型7（紫色）**: 十字形，中等，旋转+缩放动画，神秘光晕
   - **类型8（粉色）**: 心形，小尺寸，跳动动画，柔和发光

3. **粒子个体属性**（每个粒子独立）
   - **生命周期**: 粒子有出生、成长、成熟、衰老阶段
   - **能量值**: 影响发光强度和运动速度
   - **年龄**: 影响粒子大小和颜色饱和度
   - **个体随机性**: 初始属性有随机变化

4. **物理引擎**
   - 距离衰减的引力/斥力计算
   - 速度阻尼（摩擦力）
   - 边界处理（环绕或反弹）
   - 时间步进积分

5. **交互矩阵**
   - 8×8 的粒子关系矩阵
   - 可编辑的吸引/排斥强度
   - 实时更新模拟参数

### 控制面板功能
1. **基础控制**
   - 播放/暂停
   - 重置模拟
   - 粒子数量调节

2. **高级参数**
   - 引力强度全局调节
   - 作用距离调节
   - 速度限制调节
   - 阻尼系数调节

3. **矩阵编辑器**
   - 可视化关系矩阵
   - 拖拽调节相互作用
   - 预设模式切换

### 视觉效果
- **8种独特粒子形状**: 圆形、方形、星形、六边形、菱形、圆环、十字形、心形
- **个体动画效果**: 脉冲、旋转、闪烁、呼吸、流动、涟漪、缩放、跳动
- **发光效果**: 每种类型独特的发光颜色和强度
- **生命周期可视化**: 粒子大小、亮度随生命周期变化
- **能量可视化**: 高能量粒子更亮、更大
- **粒子运动拖尾**: 根据速度产生拖尾效果
- **流畅的 60fps 动画**
- **深色主题背景**: #0a0a0f

## 技术细节

### 粒子物理模型
```typescript
// 力计算
F = G * (attraction_matrix[type1][type2]) / distance^2

// 速度更新
v = v + F * dt
v = v * damping  // 阻尼

// 位置更新
p = p + v * dt
```

### 性能优化策略
- 使用空间哈希或网格加速邻近粒子查询
- 使用 Float32Array 存储粒子数据
- Canvas 批量渲染
- requestAnimationFrame 节流

### 颜色方案
- 8 种粒子颜色：红、橙、黄、绿、青、蓝、紫、粉
- 深色背景：#0a0a0f
- 发光效果：使用 canvas shadow

## 预期效果

模拟将展示以下涌现行为：
